# Space Travel - FrontEnd Mentor Challenge

- Adaptação ao Mobile:
- Tecnologias usadas:

  - React Native
  - Expo
  - Styled Components
  - React Navigation
  - JsonServer

## Instalação e Funcionamento
- Instalar as dependências:
```
yarn
```

- Rodar a API fake:
```
yarn api
```

- Rodar o projeto no app do Expo:
```
expo start
```
